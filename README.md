## インストール

[Releases](https://github.com/sevenc-nanashi/discord_activity.aux2/releases/latest)から`sevenc-nanashi.discord_activity-aux2-v{{version}}.au2pkg.zip`をダウンロードし、AviUtl2のプレビューにドラッグ&ドロップしてください。
