# discord_activity.aux2

[English](README.en.md) | [日本語](README.md)

[![AviUtl2 Catalog](https://aviutl2-catalog-badge.sevenc7c.workers.dev/badge/v/sevenc-nanashi.discord_activity-aux2)](https://aviutl2-catalog-badge.sevenc7c.workers.dev/package/sevenc-nanashi.discord_activity-aux2)

A generic plugin that updates Discord activity while editing in AviUtl2.

## Installation

Download `sevenc-nanashi.discord_activity-aux2-v{{version}}.au2pkg.zip` from [Releases](https://github.com/sevenc-nanashi/discord_activity.aux2/releases/latest) and drag and drop it into the preview of AviUtl2.
